DepAn Developer Quick Start

Experienced Eclipse and Maven practitioners should be able to get started with this abbreviated sequence:

  1. Clone the repository.
    git clone git@github.com:google/depan.git
  2. Build the libraries.
    mvn -f depan-library-master
  3. Start Eclipse.
  4. Import only the projects depan-luna-target, depan-osgi-library, and Tools.
  5. Set the target from depan-luna-target/depan-luna-target.target.
  6. Import remaining projects from DepAn repository.
    Except Test projects. Delete them from workspace.
Details for Eclipse setup, including coding conventions, etc., are included in the detailed sections.
